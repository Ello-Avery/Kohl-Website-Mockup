# Kohl's Website Mockup

## Project Overview

This project is a mockup recreation of the Kohl's e-commerce website, created as the first project for the Full Stack Development with AI for Beginners course.

## Goal

The goal of this project was to replicate the design and layout of the Kohl's homepage, focusing on:

- Navigation bar structure with search functionality
- Quick links for user account, wishlist, and shopping cart
- Secondary navigation with category links
- Responsive design elements
- Icon integration using Heroicons

## Reference Design

The mockup is based on the Kohl's website homepage, featuring:

- Black Friday deals promotions
- Product doorbusters and special offers
- Multi-tiered navigation system
- Search bar with category filtering
- Promotional banners and advertisements

## Technologies Used

- HTML5
- CSS3 (Flexbox for layout)
- SVG icons (Heroicons)
- Responsive design with media queries

## Features Implemented

- Main navigation bar with logo, search, and user actions
- Secondary navigation with store selection
- Category navigation bar
- Responsive layout that adapts to different screen sizes
- Hover effects on navigation links
- Custom styling for form inputs

## Learning Outcomes

This project helped develop skills in:

- CSS Flexbox layout techniques
- Responsive web design
- Working with SVG icons
- Structuring HTML for complex layouts
- Media queries for different screen sizes
